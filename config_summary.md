# Configuration Summary

**Environment:**  
- Python 3.11+, Flask 3.x  
- SQLite (file-based, created automatically)

**Setup:**  
```bash
pip install flask
python app.py
```

**Run:**  
Visit `http://127.0.0.1:5000/`  
Flow: Login → Upload → Prompts → Logout  

**Version Control:**  
GitHub repository with 3 branches (`backend`, `frontend`, `testing-docs`).
